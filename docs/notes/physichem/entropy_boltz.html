Macro y micro estados

En un sistema termodinámico común mente se describe los macroestados del sistema. Es decir, un conjunto de variables de estado que describen el sistema termodinámico. Por ejemplo, en un gas ideal, las variables de estado son las de presión, volumen, temperatura y cantidad de materia. Sin embargo, otra forma igual de valida de analizar el sistema es especificar que sucede en cada una de las partes que lo componen. En un gas ideal esto correspondería a especificar todas las posiciones y momentos de los átomos que componen al gas ideal en un instante, esto se conoce como microestado (Especificar el estado de cada uno de los componentes).

Para cada macroestado existen multiplies microestados, el número de microestados que le corresponden a dicho macroestado se le llama multiplicidad.

La segunda ley de la termodinámica afirma que:

Los flujos espontáneos de energía se detienen cuando el sistema está muy cerca de ó en él microestado de mayor multiplicidad.

¿Cuántos microestados hay en un macroestado?

El cálculo del número de microestados es conseguido utilizando un modelo del sistema y herramientas de probabilidad.

Cara o Cruz

La probabilidad de que un lanzamiento de moneda “justa” caiga cara o cruz es P(\text{cara}) = P(\text{cruz}) = 1/2. Para n lanzamientos existen tantos resultados (microestados) como lo permite la siguiente restricción (Suponiendo que no existe la situación en la cuál la moneda no caiga en alguna de sus dos caras) \text{número de caras} + \text{número de cruces} = n La probabilidad de que una moneda caigan exactamente y-\text{caras o } x-\text{cruces } son \begin{aligned} P(y) = P(x)-1 &= \frac{\text{\# comb. de } y}{\text{todos los resultados}} \\ \\ &= \frac{ \binom{n}{y}}{2^n} \end{aligned}

function Cg(n,r)
    if n-r < 0
        return 0
    end
    factorial(n)/(factorial(r)*factorial(n-r))
end

function Pn_take(n,y)
    Cg(n,y)/(2^n)
end

Microestados en el solido de einstein

En un solido de einstein se propone que:

  • Cada átomo en la red es un oscilador armónico cuántico tridimensional independiente.
  • Todos los átomos oscilan con la misma frecuencia.

El solido de einstein equivale a colocar q unidades de energía en N osciladores cuánticos. Estos se pueden modelar como q canicas en N cajas. La herramienta trivial es utilizar la combinatoria, pues esta se puede interpretar como contar cuantas combinaciones se pueden realizar de q elementos en N cajas.
